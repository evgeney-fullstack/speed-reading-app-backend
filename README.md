Speed Reading App Backend

REST API на Go для приложения по тренировке скорочтения.

Этот бэкенд-сервис предоставляет API для управления текстами и вопросами к ним, а также для оценки скорости чтения и понимания текста пользователями.
Функциональность

Система разделена на две роли:

    Администратор: может добавлять, редактировать и удалять тексты и вопросы к ним.

    Пользователь (клиент): получает случайный текст из базы данных, читает его, отвечает на вопросы по тексту и получает оценку по двум параметрам:

        Скорость чтения.

        Качество понимания текста (на основе ответов на вопросы).

Структура API

API построено с использованием фреймворка Gin и организовано вокруг ресурса /reading_text.

Эндпоинты для управления текстами
Метод	Путь	Описание	Требуется авторизация
POST	/reading_text/	Создание нового текста с вопросами.	Администратор
GET	/reading_text/:text_id	Получение текста и вопросов по его ID.	Пользователь / Администратор
PUT	/reading_text/:text_id	Обновление текста и/или вопросов.	Администратор
DELETE	/reading_text/:text_id	Удаление текста и связанных с ним вопросов.	Администратор

Эндпоинт для пользователя
Метод	Путь	Описание	Требуется авторизация
GET	/reading_text/random	Получение случайного текста для чтения. (Реализация ожидается)	Пользователь
